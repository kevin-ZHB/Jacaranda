输出性函数

  • return required output
  • 对原有输入不改变

    ori = ori.copy()
    ori.append('i am changed')
    return ori

print(ori)
print('\n')

全局变量 (global variable)

  • 有效范围为整个程序空间
  • 在程序的任何一个地方都可以访问这个变量，不管是在函数内部还是函数外部

Lambda

  • Lambda 是一个匿名函数，即只包括函数的 process, 但是不包括函数的名字
  • 是一种简单的、在同一行中定义函数的方法
  • lambda <参数>: <return value>
In [ ]:
def my_key(item):
    return item[::-1]

my_list = ['cz', 'bz', 'az']
print(sorted(my_list, key=my_key))
print(sorted(my_list, key=lambda item: item[::-1]))

['az', 'bz', 'cz']
['az', 'bz', 'cz']

类对象的继承
一个新的类可以继承一个父类的所有方法和属性

In [ ]:
class GC: 
    level = 1
    exp = 0
    
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def attack(self):
        print(f"{self.name} attacked!")
        
    def watch(self):
        print(f"{self.name} stares at you O.o")
        
class hop(GC):
    def jump(self):
        print(f"{self.name} jumpped!!!")

player1 = hop("john doe", 99)
player1.attack()
player1.jump()

john doe attacked!
john doe jumpped!!!
